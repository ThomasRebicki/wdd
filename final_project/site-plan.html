Overview

Purpose

One of the most common hobbies in BYUI is playing tabletop RPGs, the most popular of which being DnD. In order to play the game, usually there is a lot of “paperwork” to do. This is something that pushes away new players or players that do not have the time or energy for this extra home work. What if there was a website that could display and store documents, images and notes in an interactive way. This hub of documents could be edited and used by players, where one could set all the information they need for the game in an easy access layout. My idea is to build this hub where DnD players can post images and documents to their friends in a “newbie-friendly'' way.

Audience

This website is a tool for tabletop rpg players that aid them in world building, organization and display information to their group. This site will have simple tools and be designed primarily for new dungeon and dragon players and casual DnD groups. The aim of the website is to make the display of complex and numerous texts as simple as possible. So this website would also be perfect for players who don’t have the time or energy to dedicate for a mere game with friends.

Content

Home page

In the home page there will be a lobby, where the user will be guided on the website and choose what to do next; Instructions to world building and writing or create a character. There will also be a list of the characters the user has already created.

Images for the Home page

[Page 2]

In the second page the user will create a DnD character. In this page there will be text boxes to be filled, sites to go for extra content and at the end a compiler that will gather all the information the user inserted into a sheet.
